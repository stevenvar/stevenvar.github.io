Egalités

L'égalité des identifieurs est reflexive
Lemma eq_ident_refl :
  ∀ {A} x (e1:A) e2, ( if eq_identifier x x then e1 else e2 ) = e1.

Lemmes en rapport avec la disjonction

toute liste de variables est disjointe de la liste vide

Lemma disjoint_nil :
∀ xs,
  Disjoint_disjs xs nil.

lemme d'affaiblissement pour Disjoint

Lemma aff_disjs :
  ∀ xs a ys,
   Disjoint_disjs xs (a::ys) →
 Disjoint_disjs xs ys.

par définition deux listes disjointes n'ont pas de variables en commun
  Lemma Hh :
    ∀ xs y ys,
   Disjoint_disjs xs (y :: ys) → not (In y xs).

La disjonction est symétrique
Lemma disjs_sym :
  ∀ xs ys,
    Disjoint_disjs ys xs →
    Disjoint_disjs xs ys.

si une liste est disjointe d'une liste ys, n'importe quelle sous-liste l'est aussi
Lemma Disjoint_disjs_app_l :
  ∀ l ys r,
  Disjoint_disjs (l ++ r ) ys →
  Disjoint_disjs l ys.

même lemme que le précédent mais avec une sous liste construite dans l'autre sens
Lemma Disjoint_disjs_app_r :
  ∀ l ys r,
  Disjoint_disjs (l ++ r ) ys →
  Disjoint_disjs r ys.
